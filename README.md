## Alcance de proyecto

Nombre del proyecto: Che-planner

Descripción del proyecto:

Aplicación mobile diseñada para gestionar y dividir gastos entre grupos de personas, como amigos, familiares o compañeros de trabajo.
Permite registrar consumos compartidos, calcular automáticamente cuánto debe cada integrante y mantener un seguimiento claro de deudas y pagos dentro del grupo.
Como funcionalidad deseable, la aplicación integra una API(Azure Document Intelligence) de reconocimiento de imagen que permite escanear facturas o tickets. A partir de una imagen, el sistema identifica automáticamente los montos y conceptos, evitando la carga manual de los gastos.
El objetivo principal es simplificar la división de gastos grupales en viajes.

Requerimientos mínimos del frontend:

- Flujo de Autenticación (pantalla login y/o registro y persistencia)
- Mínimo tres (3) vistas con su ruteo y manejo de estados globales (si aplica).
- Conexión con un API externa y/o acceso a las APIs del dispositivo móvil (ejemplo geolocalización, cámara, storage, etc).

Funcionalidades incluidas:

- Pantalla Login/registro
- Pantalla Nuevo Viajes | Invitar al viaje
- Pantalla compartir con (link WhatsApp) | Añadir persona vía email
- Pantalla para cargar gasto mediante foto de Factura/Recibo
